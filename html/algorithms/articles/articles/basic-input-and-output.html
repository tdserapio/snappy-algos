Why is Input and Output even important?

Well, without taking input from the user, what data and variables would you work with? In addition, without having an output, what would be the purpose of even using the algorithm? Those two questions are the essence of why taking in input and output is essential, especially in competitive programming.

So, how do I take in Input?

Well, there are two ways to go about taking input in C++. The first way is the slow, but usually preferred way especially when time limits do not really matter. It makes use of cin, which is present in the iostream module in C++.

Slow Way (cin)

All you actually need to know for cin is that it gets all the characters until the next whitespace (space, newline, etc.), and stores it in a variable (C++ only allows for int, string, and similar data types to my knowledge). To make it simpler to understand, here is an example code snippet:


    #include <iostream>
    using namespace std;
    int main() {
        string NString;
        int NInteger;
        cout << "Input your favorite number TWICE: " << endl;
        cin >> NString;
        cin >> NInteger;
        cout << NString << " " << NInteger << endl;
    
        return 0;
    }

So as it turns out, as long as you input a numeric number (not too big either), you end up with a program that spews out said number, twice! That implies that C++ was able to "force in" and discern the fact that the random string input you just inputted was actually an integer, but could also be interpreted as a string! Amazing!

So, how do I spit out Output?

Well, just like taking in Input, Output has its own cin. Its called cout (short for console output).

Slow Way (cout)

All you actually need to know for cout is that, whatever you put in there, just gets spitted out. Only native data types work in cout, so don't even bother trying to cout an array without looping through it. Other than that, you can cout stuff like strings, which is shown in this code snippet:



    #include <iostream>
    using namespace std;
    int main() {
        cout <<"Hello World!" << endl;
        return 0;
    }
